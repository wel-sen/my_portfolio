Project Page

Casino Web App

I redesigned the sportsbook web platform to improve user navigation, simplify betting interactions, and create a scalable design system that supports a growing number of sports, markets, and user journeys.

Client

Convex Technology

Role

Product Designer

Tools Used

Figma, Adobe illustrator

Year

Jan 2023

Web design UI/UX design Design System Prototyping

Background

Convex is a digital sportbook and casino games platform provider that allows users to explore sports events, view odds, and place bets through a web-based experience.

The problem

The sportsbook platform's core functionality was solid, but its interface hadn't been refreshed in years and no longer met modern standards — reducing clarity and making it feel less engaging than contemporary betting products.
The redesign focused on modernizing the interface while preserving familiar workflows, so existing users could transition comfortably while new users got a more intuitive, visually appealing experience.

Challenges

The biggest challenge was redesigning a product with an established user base. A full overhaul risked alienating loyal users, while minor tweaks wouldn't shake the product's outdated feel. The goal was to modernize the experience without usability or breaking familiar workflows.

UI Audit

I began by reviewing the existing interface to identify areas that no longer aligned with modern design standards.

Wireframe

Low-fidelity wireframes were created to refine layouts and organize content before moving into high-fidelity designs.
